# 📝 ToDoList

A simple and efficient To-Do List web application that helps users organize and manage their daily tasks.

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Simple and user-friendly interface

## Technologies Used
- Java
- Java Servlets
- Apache Tomcat
- HTML
- CSS
- JavaScript
- SQL (MySqlWorkbench)

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/renusaklani8-tech/ToDoList.git
   ```
2. Import the project into your IDE.

3. Configure Apache Tomcat.

4. Start the Tomcat server and deploy the application.

5. Open the application in your browser.
